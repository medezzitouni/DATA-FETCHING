## How To Run the Project Locally
- install the dependencies
```bash
 npm i 
```
- Run The Developement Environment.
```bash
npm run dev
```
- Run Tests 
```bash
npm run test
```

- Run the Production  
**Run Build First**
```bash
npm run build
```
**Start the App**
```bash
npm run start
```
